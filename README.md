
## Introduction

This is a static testing grougn for bkeeping.


## Installation

Install Yeoman http://yeoman.io/

Install grunt-haml and grunt-jade locally:
$ npm install grunt-haml
$ npm install grunt-jade

execute 'yeoman server' in the project dir
